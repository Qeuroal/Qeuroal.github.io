228. Latex常见问题

发表于 2025-03-28 更新于 2025-10-02 分类于 latex
包括图片清晰度、字体缺失等问题

高质量图片

向PPT插入图像不失真

选项 -> 高级 -> 图像大小和质量 -> 不压缩

从PPT导出PDF不失真

打印 -> 选择 Adobe PDF -> 打印机属性 -> 默认设置 -> 编辑

  1. 一般

    [!TIP]
    可以设置分辨率为 2400 点/英寸

  2. 图像，所有选项全部设置为关，确定后保存

  3. 点击下方的确定或另存为，命名一下方案，然后回到打印机属性窗口，在默认设置的下拉选项中找到你命名的打印设置:

    将保存对应的设置，本机保存为 Standard(1)，下次可以直接选择该设置

[!WARNING]
建议使用 PDF/A-1b: 2005 (CMVK) 或 PDF/A-1b: 2005 (RGB).
因为在使用ieee express验证时可能会出现字体缺失问题.

裁剪PDF白色边距

方法1-adobe裁剪

导出的 PDF 有白色边距。在 Adobe Acrobat Pro DC 软件中可以裁剪。编辑PDF -> 裁剪PDF -> 选中要裁剪的内容，然后双击内容。

然后会出现如下的框，去掉勾选“显示所有框”（这一步是去掉黑边，有时会出现pdf在LaTeX里面有框的问题）然后如果内容的白边框太宽，可以点击“删除白边距”。

方法2-配置visio

先用visio画好图，然后“设计”–“大小”–“适应绘图”。visio就会自动调整画布来去除空白边缘，此时再另存为pdf即可。

去除黑边

  • 方法1: visio在另存为pdf的时候，注意去掉勾选”辅助功能文档结构标记”，这样保存的pdf在LaTeX里面就没有黑边了。

  • 方法2: 去掉勾选“显示所有框”（这一步是去掉黑边，有时会出现pdf在LaTeX里面有框的问题）

  • 方法3: 好像 这个网站 也可以！！如果没有Adobe Acrobat的朋友试试这个！

字体嵌入缺失问题

  • 方法1: 使用 PDF/A-1b: 2005 (CMVK) 或 PDF/A-1b: 2005 (RGB).

    pdf/visio使用打印导出pdf时, 默认设置设置为 PDF/A-1b: 2005 (CMVK) 或 PDF/A-1b: 2005 (RGB) 再选择”编辑”来对图片质量进行设置.

    如果需要对pdf进行裁剪, 建议使用”福昕pdf”裁剪, 裁剪完成后, 文件 -> 另存为 -> 选择保存位置 -> 保存类型改为 PDF/A (*.pdf)

  • 方法2: 用Adobe PDF印前检查可以修复
