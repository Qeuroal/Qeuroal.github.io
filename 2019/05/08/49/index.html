49. 单词拆分

发表于 2019-05-08 更新于 2021-01-07 分类于 程序设计 阅读次数：
本文字数： 645 阅读时长 ≈ 1 分钟

题目

解析

  1. 从第一个到最后一个字母进行遍历，父循环循环变量(i)决定子串的右端
  2. 其次，子循环循环变量(j)决定子串的左端
  3. 然后，若是j左侧不可拆分，那么就跳过（因为左边无法拆分，即便右边可以拆分也没有意义）
  4. 输出dpn

【释】：dp 负责记录 第n个之前（包括第n个）能否拆分
