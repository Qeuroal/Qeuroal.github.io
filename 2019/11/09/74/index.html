74. Ubuntu Server安装

发表于 2019-11-09 更新于 2025-12-08 分类于 linux
Ubuntu Server安装教程

开始安装

准备好你的机器环境并开机

基本上测试的环境只需要32GB 的硬碟空间及2GB 的记忆体就可以运作了，请依照你自己的实际环境，若是用usb 开机安装在实体机上的话，可参考这篇把ISO 档写入usb 随身碟中，本篇说明是采用PVE 虚拟机架构进行测试(**如果对PVE - Provmox VE 有兴趣的朋友，可以先参考[这里](http://blog.jason.tools/search/label/Proxmox VE))。

选择语言

以虚拟机开机后经过一连串的讯息画面后，会先出现以下选单，请利用方向键往上切换至「English」，并按下Enter 键继续安装流程。

询问是否要更新installer

接下来系统会先询问你，要不要先设定更新到最新的installer (最新的是22.06.1，现在是使用22.02.2)，如果没有特别的硬体需求或是网路暂且是中断或不稳定的情况下，请直接选择「Continue without updating」继续，因为安装完系统也会直接做升级的工作。

设定键盘Layout

接下来设定keyboard layout，没意外的话就选择「Done」采用预设值即可。

设置IP

设定IP 的部份，如果没有想特别设定的话，采用预设的DHCP 即可，不过通常伺服器应用都会设定固定IP，在此请先将光标用键盘方向键移动至「ens18 eth」这边并按下「Enter」键选取「Edit Ipv4」，再按下「Enter」键进到「IPv4 Method」的选项，预设会设定为「Automatic (DHCP)」，请先按下「Enter」键，并用方向键「下」将选项切换为「Manual」。

按下「Enter」键后，会直接出现IPv4 固定IP 的设定画面，请填入内部网路的资讯，Subnet 写法范例**( C class：192.168.1.0/24 )**。

接下来请将光标移动到「Save」，并按下「Enter」键继续下一步，会回到网路的设定画面，此时IP 已经变了，再把光标移动到「Done」按下「Enter 」键继续下一步。

指定 Proxy

接着下来看到Proxy 的设定画面，通常不会设定，直接在「Done」按下「Enter」键跳过即可。

设定预设更新来源

接下来设定预设的Ubuntu 套件来源库的位置，没有意外的话，一开始都会使用预设值，请直接在「Done」按下「Enter」键跳过即可。

分割硬碟

然后来到分割硬碟的画面，通常都是建议要自行建立硬碟分割资讯，基本上我们需要几个分割区：

  • boot - 放置核心相关档案。
  • swap - Linux 记忆体置换分割区。
  • / - root 空间，如果是测试站台，可以把剩下的空间都分给它。
  • /var - 通常会是网页、mail、Log 等资源的硬碟空间，如果是正式主机，建议将此分割区独立依据用述做更详细的规划。
  • /data - 通常会做为网站服务的用户储存空间路径，不会直接曝露在网站的连结路径中。

因为此是是示范主机，故本文只分割boot(1GB)、swap (4GB)，其它的空间就分配给/ 分割区，请先将滑鼠光标移动到「Custom storage layout」并按下「Enter」键，选完后一样把光标移动到「Done」按下「Enter」键继续。

接下来进到储存装置的设定画面，首先，先设定哪一个硬碟为「开机碟」，本文案例只有一个硬碟，请先选择硬碟的型号，按下「Enter」键，并将光标移动到「Use As Boot Device」并按下「Enter」键继续。

完成后，在下方的「USED DEVICES」列表就会出现设定好的开机磁区清单。

接下来请将光标移动到「free space」，并按下「Enter」键选择「Add GPT Partition」后，分别设定boot(1GB/Ext4)、swap(4GB/swap) 及/(约27GB/Ext4)的分割资讯，每个分割区资讯填写完成后，都请将光碟移动到「Create」按下「Enter」键完成设定。

  1. 启动（1GB）

  1. swap(4GB)

  2. /(剩下的空間 - 留空白就是把所有空間都分配下來)

  3. 分割完的资讯列表如下，确认无误后，请将光标移动到「Done」并按下「Enter」键继续。

  4. 系统会提出警告，确认后请将光碟移动到「Continue」并按下「Enter」键继续往下。

设定主机名称及使用者资讯

接下来请设定主机的名称及一位使用者帐号，此帐号未来会担负管理帐号的身份，确认后请将光标移动到「Done」并按下「Enter」键继续往下。

输入 Ubuntu Advantage token

如果您有购买Ubuntu 的订阅服务，请在此输入Token 资讯，若没有的话，请将光标移动到「Done」并按下「Enter」键继续往下。

安裝 OpenSSH Server

系统会出现安装OpenSSH Server 的画面，如果不安装，则开始时候就只能在机器前面登入，所以建议一开始请选择安装，请用键盘光标移动到「Install OpenSSH Server」的位置，按下「Enter 」键选择，「Importy SSH identity」的部份就采用预设设定即可，然后请将光标移动到「Done」并按下「Enter」键继续往下。

选择安装其它套件

Ubuntu 帮用户整合许多工具，安装流程的最后会列出清单让用户参考，这边目前建议是不需要安装，请将光标移动到「Done」并按下「Enter」键继续往下。

安装进行中

接下来系统会开始进行安装作业，如下所示，请等待其完成。

重新开机

安装完成后，画面最下的选单会变成「Reboot Now」，代表已完成安装作业，请先移除主机的光碟片，或是移除虚拟主机的开机ISO 位置设定，然后将光标移动到「Reboot Now」并按下「Enter」键重新开机即完成安装作业。
