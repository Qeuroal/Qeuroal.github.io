13. unordered_map(е“ҲеёҢиЎЁ)

еҸ‘иЎЁдәҺ 2019-01-22 жӣҙж–°дәҺ 2021-01-07 еҲҶзұ»дәҺ зЁӢеәҸи®ҫи®Ў йҳ…иҜ»ж¬Ўж•°пјҡ
жң¬ж–Үеӯ—ж•°пјҡ 1.6k йҳ…иҜ»ж—¶й•ҝ ≈ 1 еҲҶй’ҹ
unordered_mapжҳҜC++дёӯзҡ„е“ҲеёҢиЎЁпјҢеҸҜд»ҘеңЁд»»ж„Ҹзұ»еһӢдёҺзұ»еһӢд№Ӣй—ҙеҒҡжҳ е°„гҖӮ

еҹәжң¬ж“ҚдҪң

  1. еј•з”ЁеӨҙж–Үд»¶(C++11)пјҡ#include
  2. е®ҡд№үпјҡunordered_map<int,int>гҖҒunordered_map<string, double> ...
  3. жҸ’е…ҘпјҡдҫӢеҰӮе°Ҷ(вҖңABCвҖқ -> 5.45) жҸ’е…Ҙunordered_map<string, double> hashдёӯпјҢhash[вҖңABCвҖқ]=5.45
  4. жҹҘиҜўпјҡhash[вҖңABCвҖқ]дјҡиҝ”еӣһ5.45
  5. еҲӨж–ӯkeyжҳҜеҗҰеӯҳеңЁпјҡhash.count("ABC") != 0 жҲ– hash.find("ABC") != hash.end()

йҒҚеҺҶ

иҝӣйҳ¶ж“ҚдҪң

еҰӮжһңжғіи®©иҮӘе®ҡд№үзҡ„classдҪңдёәkey(unordered_map<key,value>)жқҘдҪҝз”Ёunordered_mapпјҢйңҖиҰҒе®һзҺ°пјҡ

  1. е“ҲеёҢеҮҪж•°пјҢйңҖиҰҒе®һзҺ°дёҖдёӘclassйҮҚиҪҪoperator()пјҢе°ҶиҮӘе®ҡд№үclassеҸҳйҮҸжҳ е°„еҲ°дёҖдёӘsize_tзұ»еһӢзҡ„ж•°гҖӮдёҖиҲ¬еёёз”Ёstd::hashжЁЎжқҝжқҘе®һзҺ°гҖӮ
  2. еҲӨж–ӯдёӨдёӘиҮӘе®ҡд№үclassзұ»еһӢзҡ„еҸҳйҮҸжҳҜеҗҰзӣёзӯүзҡ„еҮҪж•°пјҢдёҖиҲ¬еңЁиҮӘе®ҡд№үclassйҮҢйҮҚиҪҪoperator==гҖӮ
