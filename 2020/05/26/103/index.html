进入tmux翻屏模式

  • 先按 ctrl+b，松开，然后再按 [
  • 实现上下翻页: 进入翻屏模式后，PgUp, PgDn 实现上下翻页
  • 退出: q

ps: ctrl + b + [，很多人会以为是同时按三个键，但是这样是不行的。

简单介绍下tmux的其他功能

常用快捷键

快捷键 功能
Ctrl-B % 竖直拆分屏幕（两个 Shell 分别位于左右）
Ctrl-B " 水平拆分屏幕（两个 Shell 分别位于上下）
Ctrl-B O 切换到另一个 Shell
Ctrl-B ? 查看帮助
Ctrl-B d 断开 tmux 并让其在后台运行（可以使用 tmux attach 重新进入）
